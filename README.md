# The Void Lands - Source Scripts

Welcome to the standalone script repository for **Void Lands**. 

This repository was created specifically to provide potential employers, recruiters, and technical reviewers with fast,frictionless access to the game's core architecture and codebase without requiring them to download gigabytes of heavy art assets, audio files, or engine binaries.

---

## 📌 Repository Purpose
* **Easy Code Review:** A lightweight, pure-code environment designed for rapid reading, code-quality evaluation, and architectural review.
* **Production-Ready Implementation:** Contains all functional gameplay systems, systems architecture, and utility scripts running the mechanics of *Void Lands*.
* **Unity-Ready Metadata:** Includes corresponding `.meta` files to preserve structural references and inspector assignments for reference.

---

## 🛠️ Tech Stack & Systems Architecture
* **Engine:** Unity
* **Language:** C#

---

## 🔒 Project Notice
This repository contains **source code only**. Because all asset files (3D models, textures, animations, audio) have been stripped out to ensure a lightweight download, this repository cannot be built or run as a standalone Unity project.
